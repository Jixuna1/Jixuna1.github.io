# Portfolio Gamer - BTS SIO SLAM

Ce portfolio a été conçu avec un thème "Jeu Vidéo Moderne" / Cyberpunk, entièrement en HTML, CSS et JavaScript Vanilla (sans frameworks lourds) pour garantir une performance maximale et une modification facile.

## 🚀 Comment l'utiliser ?

1.  Ouvrez simplement le fichier `index.html` dans votre navigateur (Double-clic).
2.  Vous verrez l'animation d'introduction et pourrez naviguer via le "HUD".

## 🛠️ Personnalisation

### Changer les textes
Ouvrez `index.html` avec un éditeur de texte (VS Code, Notepad++, etc.) et modifiez les contenus textuels (Vos informations, vos projets).

### Changer l'Avatar
Remplacez l'image dans `assets/avatar.png` par votre propre photo (gardez un format carré pour un meilleur rendu).

### Ajouter des projets
Dans la section `#quests`, dupliquez un bloc `<div class="quest-card">` et modifiez le titre, la description et les tags.

## 🎨 Design System
Les couleurs et polices sont définies dans `css/style.css` sous `:root`.
- `--neon-cyan`: Couleur principale
- `--neon-purple`: Couleur secondaire
- `--bg-dark`: Fond

Amusez-vous bien !
